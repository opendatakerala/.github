# Open Data Kerala

Open Data Kerala is an open community initiative spearheaded by the Sahya Digital Conservation Foundation, collaborating with:

- OpenStreetMap Kerala
- Wikimedians of Kerala User Group
- Swathandra Malayalam Computing
- CODD-Keralam
- Geominds
- Kerala Biodiversity Monitoring Network
- CitizenScience Kerala
